1. Input data for the all the files in programCode is available in the
   folder "InputData"
2. Copy the files  from "InputData" to appropriate source folder
3. To run code go to the folder "ProgramCode"
   DBSCAN: execute the file dbscan.scala          Input Files: tripdata_1.gsd, tripfare_1.gsd
   KNearest: execute the file knearest.scala      Input Files: tripdata_1.gsd, tripfare_1.gsd,tripdata_2.gsd,                                                                         tripfare_2.gsd
   Naive Bayes: execute the file naivebayes.scala
